# Getting Started
- Install `rvm`
- Run `rvm install ruby-3.0.0`
- Run `rvm use`
- Run `gem install rails`
- `cd` to cvwo-winter-rails-with-auth directory 
- Run `bundle install`
- `rails db:create`
- `rails db:migrate`
- `rails db:seed`
- Run `rails s` to start the app locally
- Copy link and paste it into react misc/constants.tsx, database variable
